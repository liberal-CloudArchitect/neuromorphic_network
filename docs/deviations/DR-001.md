# P3 task-score 与 OOD 主指标不完全可计算

正式产物审查发现，`p3-protocol-v2` 要求三任务 ID score 先按 chance 标准化，
但没有冻结 SmallGraph rollout success 的 chance 定义。SmallGraph OOD 产物只包含
`optimal_action_rate`，没有 OOD rollout `success_rate`。因此 task-score 与 OOD 两类
confirmatory 比较不能按预注册定义通过。

报告保留 raw task-score 与 `optimal_action_rate` OOD proxy 的配对 bootstrap，明确标记为
`INVALID/FAIL`，不用于授予收益类别。不得在查看结果后补写 chance 或用 proxy 替代主指标；
若后续继续研究，应建立新 protocol、重新生成所有模型的对称证据。
