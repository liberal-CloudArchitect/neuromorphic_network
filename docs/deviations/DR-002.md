# Forgetting 收益缺少冻结的最小效应阈值

`p3-protocol-v2` 冻结了 forgetting 的计算方向和双主基线 family，却没有冻结该类别的
最小工程效应。因此正式报告计算并披露 `baseline - modular`、95% CI、原始 p 与 Holm p，
但将类别标记为 `INVALID/FAIL`，不允许仅凭显著性事后认定通过。

本偏差不删除任何实验，也不改变 `GATE-3` 的完整性判断；它会阻止 forgetting 成为
`GATE-NN-MVP` 的通过类别。后续阈值必须在新 protocol 查看新数据前冻结。
