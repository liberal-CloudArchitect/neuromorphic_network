# CR-002：以归一化学习曲线 AULC 替代未定义达标样本数

## 变更

P0 文档冻结了“达标样本数减少”的方向和 15% 总体阈值，但没有冻结各任务达标分数，因此该指标无法在查看正式曲线前无歧义执行。P3 在任何正式 test/OOD 运行前改用 analysis split 上的归一化 AULC：网络总体收益阈值为相对主基线提高至少 15%；predictive retrained ablation 的模块门槛为 AULC 相对恶化至少 10%。

## 约束

- analysis split 不参与 pilot 选择、早停或 checkpoint 选择。
- 旧 P1 报告保持原样，不回写或重新解释。
- 正式结果产生后不得恢复或重新定义 `N@tau`；如需比较必须建立新 protocol。

## 影响

同步更新统计协议、实验协议、P3 配置、报告 schema 和 Gate claim matrix。该变更不调整 task score、OOD、forgetting、稀疏收益或三个模块因果门槛。
