# 科学边界与声明

> 本文冻结项目的科学表述边界。`ACCEPTED` 表示该边界必须约束实现、报告与展示；`GATE-0` 是否通过仍由独立评审报告裁决。

截至 P3 正式裁决，`GATE-3 PASSED` 只表明人工模型实验完整，`GATE-NN-MVP FAILED`
表明当前实现没有达到预注册网络收益条件。情景记忆与工作记忆在合成任务中的消融效应
不能解释为海马、前额叶或其他脑组织的机制验证。

## 项目主张

本项目研究的是**受神经科学启发、通过任务与因果实验验证的模块化神经网络**。核心对象是感觉编码、情景记忆、工作记忆、预测适配、动作选择和稀疏路由六类计算模块，以及它们的训练、通信、状态与损失机制。

## 可以声称的内容

- 某项神经科学证据启发了一个明确、可否证的工程假说。
- 某个模块在冻结协议下相对公平基线产生了可重复的任务、泛化、样本效率、遗忘或计算收益。
- 消融、替换或扰动改变了**人工模型内部**的行为或表示。
- telemetry 描述人工网络的计算活动、路由、学习与误差信号。

## 不可以声称的内容

- 网络复刻了真实人脑、意识、认知主体或临床神经活动。
- 模块名称与真实脑区存在一一对应关系。
- 感觉编码、记忆、预测、选择或路由是某个脑区的唯一功能，或某个脑区只承担一个模块功能。
- 高激活、注意力权重或梯度天然等于重要性或真实脑区因果贡献。
- 3D 展示是 BOLD、神经放电或脑成像数据。
- 未经公平基线、多种子和预注册统计验证的结果证明“类脑”优越性。

## 工程边界

- 3D 模型是可关闭的 telemetry 消费者，不参与 forward、loss、优化、路由或决策。
- 网络训练和评估不得依赖 atlas、viewer、Web 服务或可视化库。
- telemetry 关闭后必须保持模型语义不变；性能开销单独报告。
- `BrainPacket`、`ModuleOutput`、状态与损失契约由 P0 公共契约冻结；后续实现不得为了 3D 展示增加模型必需字段或改变计算语义。
- 六模块注册名称只标识人工计算职责。证据注册表中的脑区、物种和测量不能被转换为结构同一性、机制同一性或临床解释。

## 证据与结论纪律

- 报告必须保持 `Evidence → Abstraction → Hypothesis → Test → Result` 链路，并同时陈述来源局限和工程类比距离。
- 神经科学来源提供设计动机，不是模型有效性的结果；模型主张只能由冻结任务、公平基线、多 seed 和预注册统计支持。
- 相关性 telemetry、probe、注意力或路由权重只能描述人工模型内部关联；因果用语必须来自预注册消融、替换或扰动。
- 未达否证门槛的模块应缩减、替换或删除。不得通过改名、挑选 seed、替换指标或视觉叙事保留失败主张。

## 声明模板

研究报告与演示必须包含：

> 本系统展示的是人工模型活动遥测，不是 BOLD、真实神经活动或临床结论。脑功能名称仅描述工程启发来源；所有因果结论仅适用于被测试的人工模型与冻结实验协议。

## 退出条件

若模块不能相对参数匹配主基线产生预注册收益，或因果实验不能支持其功能分工，应缩减、替换或删除该模块，不得以 3D 演示效果替代模型证据。
