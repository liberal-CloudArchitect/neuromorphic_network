## 评审检查

- [ ] 六个 MVP 模块的每项主张均有至少一条可定位来源。
- [ ] 每条来源记录研究对象、任务、主要结果与局限，避免只记录标题。
- [ ] 每条抽象说明从生物机制到计算机制的简化步骤。
- [ ] 每个假说都有方向性指标、比较对象和明确否证条件。
- [ ] 编码器与选择器具有容量匹配替换/扰动对照；至少三个关键模块具有任务级消融。
- [ ] 评审人、日期、commit hash 和裁决已记录。
